# QuantexQ

Industrial drilling operations dashboard built with Vite, React, TypeScript, and shadcn-ui. Provides KPI cards, pressure charts, pump status, and notifications in a dark monitoring-friendly layout.

## Getting started

```sh
npm install
npm run dev
```

Visit http://localhost:5173 to view the app.

## Scripts

- `npm run dev` — start the dev server
- `npm run build` — production build
- `npm run preview` — preview the production build
- `npm run lint` — run ESLint
- `npm run test` — run tests once
- `npm run test:watch` — run tests in watch mode

## Tech stack

- Vite + React + TypeScript
- Tailwind CSS + shadcn-ui components
- React Router, TanStack Query, Recharts
